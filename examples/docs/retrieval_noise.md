# 相似但非标准来源的检索噪声

这份文档用于构造检索噪声。它包含 RAG、top-k、chunk size、上下文、延迟、检索、引用等词，但不是示例问题集的标准答案来源。

在真实 RAG 系统里，知识库中经常会存在主题相近但并不直接回答问题的文档。检索器可能把这些相似文档一起返回，导致 top-k 结果中同时包含有用上下文和噪声。

top-k 变大时，系统返回的候选片段数量更多。这样可能提高召回，也可能让无关 chunk 进入上下文，增加模型阅读负担、上下文成本和响应延迟。

chunk size 过大时，一个片段可能同时包含多个主题；chunk size 过小时，片段可能缺少必要上下文。两种情况都可能影响最终回答质量。

这份文档不应该作为 `rag_basics.md` 问题的标准引用来源。它的作用是帮助 precision@k 暴露检索结果中的噪声。
